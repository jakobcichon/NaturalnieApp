DYMO Label Framework JavaScript Library Samples: Printers and Multiple Labels Printing

This sample shows how new DYMO Label Framework JavaScript Library can be used to enumerate all installed DYMO printers along with their characteristics like online/offline status, printer type, connecting type, etc.

Try to connect/disconnect printers and click on 'Refresh' button to see the changes in the table. Notes:

  • connection state detection works only for local printers
  • isConnected property is not updated automatically. To get current connection state it is nessesary to call dymo.label.framework.getPrinters() again
